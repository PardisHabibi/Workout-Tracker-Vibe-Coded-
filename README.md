# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/drive/1x46X-zjl4RlMblfMI1QuhyfSfKR1U8hd

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`


## Functions

- Track workouts with date and time
- Add your own exercises
- Create programs to start when selected
- Add notes to each set
- Built in timer
- Check and edit past workouts
